# 🌐 Developer Portfolio Website

A professional, fully responsive, and modern developer portfolio website designed to showcase my skills, projects, and experiences. Built with a sleek dark-themed UI and smooth interactive elements.

🚀 **Live Link:** [View Live Site](https://mdkamran623.github.io/Portfolio)

---

## ✨ Features

- 📱 **Fully Responsive Design:** Optimised for all screen sizes (Mobile, Tablet, and Desktop).
- 🎨 **Modern Dark UI:** Beautiful aesthetics with pixel-perfect spacing, fluid typography, and dynamic hover effects.
- ⚡ **Interactive Sections:** Clean navigation bar, dynamic skill bars, project showcasing with category filters, and a functional contact form.
- 📬 **Get In Touch:** Seamlessly integrated contact section to make it easy for recruiters and clients to reach out.

---

## 🛠️ Technical Arsenal (Tech Stack)

This portfolio is crafted using pure, high-performance web technologies:

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Icons & Graphics:** Font Awesome / SVG
- **Deployment:** Netlify

2. **Navigate into the directory:**
   ```bash

   ```
3. **Open the project:**
   Simply open `index.html` in your favorite web browser or use the **Live Server** extension in VS Code.
